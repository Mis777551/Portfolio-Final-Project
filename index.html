World clock

I developed a responsive and interactive world clock application, showcasing my expertise in modern front-end technologies and precise time-based logic. I implemented elegant layout techniques with CSS Flexbox/Grid to ensure the design adapts fluidly to desktop and mobile screens, and integrated intuitive UI controls—like adding or removing clocks—which demonstrates my attention to user experience and state management. This project reflects my ability to fuse functional programming, responsive design principles, and interactive interface elements into a polished, user-centric application—skills I’m eager to bring to your team.
